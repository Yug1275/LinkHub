# LinkHub

LinkHub is a customizable web dashboard that lets users organize and access their favorite websites from a single place.
It provides a clean interface to manage links, categories, widgets, and personalized dashboard settings.

## Features

* User authentication (Signup / Login)
* Personal dashboard for managing websites
* Create and manage categories
* Drag & drop website arrangement
* Grid and category view
* Command palette search (Ctrl + K)
* Dashboard widgets (Clock, Notes, Weather, Pomodoro)
* Theme customization (Dark / Light / Auto)
* Custom backgrounds (image, gradient, opacity control)
* Import / Export dashboard configuration
* Focus mode for distraction-free usage
* Fully responsive design

## Tech Stack

Frontend:

* React
* Tailwind CSS
* Framer Motion
* React DnD

Backend:

* Node.js
* Express.js

Database:

* MongoDB Atlas
* Mongoose
